## 1. アプリ概要

「はむはむ画像圧縮くん」は、JPEG・PNG・PDF・Office ファイルをドラッグ＆ドロップするだけで、**最新の圧縮アルゴリズム**により高品質・高圧縮を実現するデスクトップアプリケーションです。WebP・AVIF・JPEG XL への次世代フォーマット変換にも対応しています。

外部ツール（Ghostscript、pngquant、cjpegli、ECT、cwebp、cjxl、avifenc）はすべてアプリに**同梱**されており、追加のインストールは一切不要です。

**CLI（コマンドライン）モード** も搭載しており、バッチ処理やCI/CDパイプラインからも利用できます。

## 3. 対応ファイル形式

### 入力形式

| 形式 | 拡張子 | 圧縮方式 |
|---|---|---|
| **JPEG** | `.jpg`, `.jpeg` | Jpegli (Lossy) |
| **PNG** | `.png` | pngquant (Lossy減色) + Oxipng (最適化) + ECT (Lossless再圧縮) |
| **PDF** | `.pdf` | Ghostscript (画像ダウンサンプリング + JPEG再圧縮) |
| **Office** | `.docx`, `.xlsx`, `.pptx` | 内部画像を個別に最適化（JPEG/PNG） |

### 出力変換形式

| 変換先 | ツール | 特徴 |
|---|---|---|
| **WebP** | cwebp | JPEG/PNGより20-30%小さい。主要ブラウザ全対応 |
| **AVIF** | avifenc | Web利用可能な形式で最高の圧縮率 |
| **JPEG XL** | cjxl | JPEGの正式後継規格。ロスレスJPEG再圧縮が可能 |

## 4. 圧縮アルゴリズム詳細

### 4.1 JPEG 圧縮 — Jpegli

| 項目 | 内容 |
|---|---|
| **ツール** | [cjpegli](https://github.com/google/jpegli) (libjxl プロジェクト内) |
| **開発元** | Google |
| **特徴** | 従来の JPEG と完全互換。最新の心理視覚モデルにより、同じファイルサイズでも従来の libjpeg-turbo より高い画質を実現 |
| **デフォルト品質** | 85% (設定画面で 10–100% に調整可能) |
| **プログレッシブ** | デフォルト有効（`-p 2`フラグ） |

```
入力 JPEG → cjpegli (品質パラメータ指定) → 出力 JPEG
```

### 4.2 PNG 圧縮 — pngquant + Oxipng + ECT（3段階最適化）

| ステップ | ツール | 種別 | 効果 |
|---|---|---|---|
| 1️⃣ 減色 | [pngquant](https://pngquant.org/) | Lossy | 色数を削減し大幅にサイズダウン (品質60-95%) |
| 2️⃣ 最適化 | [Oxipng](https://github.com/shssoichiro/oxipng) | Lossless | 不要チャンク削除 + フィルター最適化 + アルファ最適化 |
| 3️⃣ ロスレス再圧縮 | [ECT](https://github.com/fhanau/Efficient-Compression-Tool) v0.9.5 | Lossless | Deflate ストリームを最適化しさらに数%〜十数%削減 |

```
入力 PNG → pngquant (--quality=60-95, --colors=N) → Oxipng (strip + filter) → ECT (-9 --strict) → 出力 PNG
```

> [!TIP]
> この3段階パイプラインは、PNG最適化の**世界最高水準**のワークフローです。

### 4.3 PDF 圧縮 — Ghostscript

| 項目 | 内容 |
|---|---|
| **ツール** | [Ghostscript](https://www.ghostscript.com/) |
| **方式** | PDF 内の画像をダウンサンプリング + JPEG 再圧縮 |
| **プリセット** | 5段階（高品質 460dpi 〜 最小 72dpi） |
| **推奨設定** | 235 dpi / JPEG Q82（⭐推奨） |
| **サイズ目標** | 目標KB指定で自動バイナリサーチ |

```
入力 PDF → Ghostscript (pdfwrite + ダウンサンプル + JPEG圧縮) → 出力 PDF
```

#### PDF 圧縮プリセット一覧

| プリセット | 解像度 (DPI) | JPEG品質 | 用途 |
|---|---|---|---|
| 高品質 | 460 dpi | Q90 | 印刷用 |
| ⭐ 推奨 | 235 dpi | Q82 | 一般配布用 |
| 標準 | 200 dpi | Q82 | Web掲載用 |
| 軽量 | 150 dpi | Q75 | メール添付用 |
| 最小 | 72 dpi | Q70 | プレビュー用 |

### 4.4 WebP 変換 — cwebp

| 項目 | 内容 |
|---|---|
| **ツール** | [cwebp](https://developers.google.com/speed/webp) (libwebp) |
| **開発元** | Google |
| **特徴** | JPEG/PNG 両方の用途をカバー。主要ブラウザすべてで対応済み |

```
入力 JPEG/PNG → cwebp (-q 品質) → 出力 WebP
```

### 4.5 AVIF 変換 — avifenc

| 項目 | 内容 |
|---|---|
| **ツール** | [avifenc](https://github.com/AOMediaCodec/libavif) (libavif) |
| **特徴** | AV1ベースの画像フォーマット。Web利用可能な形式の中で最高の圧縮率 |
| **速度設定** | speed 6 (バランス型) |

```
入力 JPEG/PNG → avifenc (-q 品質, -s 6) → 出力 AVIF
```

### 4.6 JPEG XL 変換 — cjxl

| 項目 | 内容 |
|---|---|
| **ツール** | [cjxl](https://github.com/libjxl/libjxl) (libjxl) |
| **特徴** | JPEGの正式後継規格。既存のJPEGをロスレスで再圧縮できる唯一のフォーマット |
| **モード** | ロスレス (JPEG→JXL 完全復元可能) / Lossy (品質指定) |

```
# ロスレス JPEG → JXL（ビット完全復元可能）
入力 JPEG → cjxl → 出力 JXL

# Lossy
入力 JPEG/PNG → cjxl (-q 品質) → 出力 JXL
```

### 4.7 おまかせ機能 — SSIM + CIEDE2000 自動品質

| 項目 | 内容 |
|---|---|
| **方式** | バイナリサーチで「劣化が目立たない限界」を自動検出 |
| **JPEG 判定基準** | SSIM ≥ 0.95 |
| **PNG 判定基準** | SSIM ≥ 0.98 AND 平均ΔE ≤ 1.5 AND 最大ΔE ≤ 5.0 |
| **ΔE 計算** | CIEDE2000 (ISO/CIE 11664-6:2014) — 人間の知覚に近い色差評価 |

```
品質 高(95)→低(30) の方向でバイナリサーチ
→ 各品質でエンコード → SSIM/ΔE を計算
→ 閾値を満たす最低品質を採用（最大の削減率）
```

### 4.8 Office ファイル圧縮

| 項目 | 内容 |
|---|---|
| **対応形式** | `.docx`, `.xlsx`, `.pptx` |
| **方式** | ZIP展開 → 内部画像を個別に JPEG/PNG 圧縮 → 再ZIP |

## 7. CLI（コマンドライン）モード

GUI とは別に、ターミナルから直接圧縮を実行できる `hamham-compress` コマンドを提供しています。

### 使用例

```bash
# JPEG 圧縮
hamham-compress -q 85 -o ./compressed/ photo.jpg

# フォルダ一括処理
hamham-compress -o ./out/ ./images/

# おまかせモード
hamham-compress --auto-quality photo.jpg

# WebP 変換
hamham-compress --webp -q 80 input.png

# JPEG XL ロスレス変換
hamham-compress --jxl photo.jpg
```

### オプション一覧

| オプション | 短縮 | デフォルト | 説明 |
|---|---|---|---|
| `--quality` | `-q` | 85 | JPEG/WebP/AVIF 品質 (10-100) |
| `--png-colors` | `-c` | 256 | PNG 色数 (2-256) |
| `--pdf-dpi` | | 235 | PDF 解像度 |
| `--pdf-quality` | | 82 | PDF 内 JPEG 品質 |
| `--output` | `-o` | `~/Desktop/圧縮済み/` | 出力ディレクトリ |
| `--webp` | | false | WebP 変換モード |
| `--avif` | | false | AVIF 変換モード |
| `--jxl` | | false | JPEG XL 変換モード |
| `--jxl-lossless` | | true | JXL ロスレスモード |
| `--auto-quality` | `-a` | false | おまかせモード |
| `--target-size` | `-s` | 0 | 目標サイズ (KB, PDF用) |
| `--max-width` | `-W` | 0 | 最大幅 (0=無制限) |
| `--max-height` | `-H` | 0 | 最大高さ (0=無制限) |
| `--no-progressive` | | false | プログレッシブ JPEG を無効化 |
| `--keep-metadata` | | false | メタデータ削除を無効化 |

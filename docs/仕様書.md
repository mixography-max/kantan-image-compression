## 1. アプリ概要

「はむはむ画像圧縮くん」は、JPEG・PNG・PDF ファイルをドラッグ＆ドロップするだけで、**最新の圧縮アルゴリズム**により高品質・高圧縮を実現するデスクトップアプリケーションです。

外部ツール（Ghostscript、pngquant、cjpegli、ECT）はすべてアプリに**同梱**されており、追加のインストールは一切不要です。

## 3. 対応ファイル形式

| 形式 | 拡張子 | 圧縮方式 |
|---|---|---|
| **JPEG** | `.jpg`, `.jpeg` | Jpegli (Lossy) |
| **PNG** | `.png` | pngquant (Lossy減色) + ECT (Lossless再圧縮) |
| **PDF** | `.pdf` | Ghostscript (画像ダウンサンプリング + JPEG再圧縮) |

---

## 4. 圧縮アルゴリズム詳細

### 4.1 JPEG 圧縮 — Jpegli

| 項目 | 内容 |
|---|---|
| **ツール** | [cjpegli](https://github.com/google/jpegli) (libjxl プロジェクト内) |
| **開発元** | Google |
| **特徴** | 従来の JPEG と完全互換。最新の心理視覚モデルにより、同じファイルサイズでも従来の libjpeg-turbo より高い画質を実現 |
| **デフォルト品質** | 85% (設定画面で 10–100% に調整可能) |

```
入力 JPEG → cjpegli (品質パラメータ指定) → 出力 JPEG
```

### 4.2 PNG 圧縮 — pngquant + ECT（2段階最適化）

| ステップ | ツール | 種別 | 効果 |
|---|---|---|---|
| 1️⃣ 減色 | [pngquant](https://pngquant.org/) | Lossy | 色数を削減し大幅にサイズダウン (品質60-95%) |
| 2️⃣ ロスレス再圧縮 | [ECT](https://github.com/fhanau/Efficient-Compression-Tool) v0.9.5 | Lossless | Deflate ストリームを最適化しさらに数%〜十数%削減 |

```
入力 PNG → pngquant (--quality=60-95, --colors=N) → ECT (-9 --strict) → 出力 PNG
```

> [!TIP]
> この2段階パイプラインは、PNG最適化の**世界最高水準**のワークフローとして広く知られています。

### 4.3 PDF 圧縮 — Ghostscript

| 項目 | 内容 |
|---|---|
| **ツール** | [Ghostscript](https://www.ghostscript.com/) |
| **方式** | PDF 内の画像をダウンサンプリング + JPEG 再圧縮 |
| **プリセット** | 5段階（高品質 460dpi 〜 最小 72dpi） |
| **推奨設定** | 235 dpi / JPEG Q82（⭐推奨） |

```
入力 PDF → Ghostscript (pdfwrite + ダウンサンプル + JPEG圧縮) → 出力 PDF
```

#### PDF 圧縮プリセット一覧

| プリセット | 解像度 (DPI) | JPEG品質 | 用途 |
|---|---|---|---|
| 高品質 | 460 dpi | Q90 | 印刷用 |
| ⭐ 推奨 | 235 dpi | Q82 | 一般配布用 |
| 標準 | 200 dpi | Q82 | Web掲載用 |
| 軽量 | 150 dpi | Q75 | メール添付用 |
| 最小 | 72 dpi | Q70 | プレビュー用 |

## 5. 同梱ツール一覧

すべてアプリ内にバンドルされ、ユーザーの環境構築は不要です。

| ツール | バージョン | macOS 取得方法 | Windows 取得方法 |
|---|---|---|---|
| **cjpegli** | libjxl v0.11.2 | ソースからビルド (cmake + ninja) | libjxl GitHub Release (static) |
| **pngquant** | Latest | Homebrew | Chocolatey |
| **ECT** | v0.9.5 | 公式 GitHub Release | 公式 GitHub Release |
| **Ghostscript** | Latest | Homebrew (+dylibs +リソース) | Chocolatey (+DLL +リソース) |

### ランタイムパス解決の優先順位

```
1. バンドルされたバイナリ（アプリ内 resources/tools/）
2. システムにインストール済みのバイナリ（PATH検索）
```

### 環境変数の自動設定

| 変数 | 対象 | 内容 |
|---|---|---|
| `DYLD_LIBRARY_PATH` | macOS | バンドルされた `.dylib` のパス |
| `PATH` (追加) | Windows | バンドルされた `.dll` のパス |
| `GS_LIB` | 全OS | Ghostscript リソース（フォント、Init等）のパス |

## 6. 機能一覧

### 基本機能
- ✅ ドラッグ＆ドロップによるファイル圧縮
- ✅ 複数ファイルの一括圧縮
- ✅ 圧縮結果の表示（元サイズ → 圧縮後サイズ、削減率）
- ✅ 保存先フォルダの選択・永続化

### 設定項目
- JPEG 品質 (10–100%)
- PNG 色数 (2–256)
- PDF プリセット選択 / DPI / JPEG品質 個別設定
- Office 内画像 JPEG 品質
- ファイルをフォルダにまとめるオプション

### UI
- 🐹 ハムスターマスコットキャラクター
- パステルカラーのあたたかいデザイン
- アニメーション付きドロップゾーン
- 圧縮アルゴリズム解説パネル
